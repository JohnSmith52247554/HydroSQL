The document UserConfig.bin is a two dimension sheet. username1 username2 username3 ... password_hash1 password_hash2 password_hash3 ... table_nameA auth_levelA1 auth_levelA2 auth_levelA3 table_nameB auth_levelB1 auth_levelB2 auth_levelB3 table_nameC auth_levelC1 auth_levelC2 auth_levelC3 ...

When saving, the sheet will be save in a linear structure as following. USERNAME_SIZE username1, username2, username3, ... password_hash1, password_hash2, password_hash3, ... TABLE_NAME_SIZE table_nameA, table_nameB, table_nameC, ... auth_levelA1, auth_levelA2, auth_levelA3, ... auth_levelB1, auth_levelB2, auth_levelB3, ... auth_levelC1, auth_levelC2, auth_levelC3, ... ...
